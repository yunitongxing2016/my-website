# 我的个人主页

这是一个适合学习网页制作的个人网站，可以直接发布到 GitHub Pages。

## 网站内容

- 个人主页介绍
- 全国城市演示天气卡片
- 学习日志
- 点赞和评论
- 猜数字小游戏

## 文件说明

- `index.html`：网站首页
- `styles.css`：网站样式
- `script.js`：首页互动、学习日志、点赞评论
- `weather.js`：省份城市列表和演示天气数据
- `game.html`：猜数字小游戏页面
- `game.js`：小游戏逻辑

## 重要提醒

学习日志、点赞、评论使用浏览器本地保存。发布到互联网后，每个访客的内容会保存在自己的浏览器里，不会自动汇总到同一个服务器。

如果以后想做真正的公共博客评论区，需要再加入后端和数据库。
